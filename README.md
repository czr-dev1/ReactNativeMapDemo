# ReactNativeMapDemo
All of the important code is inside the `app` directory

## Install Expo CLI
`npm install --global expo-cli`

## Install Packages (Should add all the packages you need)
`expo install` or

`npm install`

## Run
Start Metro Bundler

`expo start` or

`npm start`

Start App on Android
`npm run android`

More Commands Here:
https://docs.expo.io/workflow/expo-cli/
